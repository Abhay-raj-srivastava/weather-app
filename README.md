# Weather App

A responsive Weather App built using HTML, CSS, and JavaScript.

## Features
- Search weather by city
- Real-time weather data using OpenWeatherMap API
- Dynamic weather icons
- Temperature, humidity, and wind speed
- Enter key support
- Loading state
- Error handling

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API
